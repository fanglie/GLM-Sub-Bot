## 🔍 日志分析工具

### Windows PowerShell 命令

```powershell
# 1. 查看最新日志（最后 100 行）
Get-Content logs\auto-buy_*.log -Tail 100

# 2. 查看所有错误
Select-String -Path logs\auto-buy_*.log -Pattern "\[ERROR\]" -Context 3

# 3. 查看库存状态变化
Select-String -Path logs\auto-buy_*.log -Pattern "库存状态"

# 4. 查看点击尝试
Select-String -Path logs\auto-buy_*.log -Pattern "第.*次尝试"

# 5. 查看成功操作
Select-String -Path logs\auto-buy_*.log -Pattern "\[SUCCESS\]"

# 6. 实时查看日志（脚本运行时）
Get-Content logs\auto-buy_*.log -Tail 20 -Wait

# 7. 查看网络请求
Select-String -Path logs\auto-buy_*.log -Pattern "\[NETWORK\]"

# 8. 查看登录状态
Select-String -Path logs\auto-buy_*.log -Pattern "\[AUTH\]"

# 9. 打开截图目录
explorer screenshots

# 10. 导出错误到文件
Select-String -Path logs\auto-buy_*.log -Pattern "\[ERROR\]" -Context 3 > errors.txt
```

### Linux/Mac 命令

```bash
# 1. 查看最新日志
tail -n 100 logs/auto-buy_*.log

# 2. 查看所有错误
grep "\[ERROR\]" logs/auto-buy_*.log -A 3

# 3. 查看库存状态
grep "库存状态" logs/auto-buy_*.log

# 4. 实时查看日志
tail -f logs/auto-buy_*.log

# 5. 打开截图目录
open screenshots  # Mac
xdg-open screenshots  # Linux
```

## 🛠️ 故障排查工具

### 快速诊断脚本

创建 `quick-diagnose.ps1`:
```powershell
Write-Host "=== 智谱 AI 抢购日志诊断工具 ===" -ForegroundColor Cyan

# 1. 找到最新日志
$latestLog = Get-ChildItem logs\auto-buy_*.log | Sort-Object LastWriteTime -Descending | Select-Object -First 1
if ($latestLog) {
    Write-Host "`n最新日志文件：$($latestLog.Name)" -ForegroundColor Green
    
    # 2. 统计日志数量
    $totalLines = (Get-Content $latestLog.FullName | Measure-Object -Line).Lines
    Write-Host "总行数：$totalLines" -ForegroundColor Green
    
    # 3. 统计错误数量
    $errorCount = (Select-String -Path $latestLog.FullName -Pattern "\[ERROR\]").Count
    Write-Host "错误数量：$errorCount" -ForegroundColor $(if ($errorCount -gt 0) {"Red"} else {"Green"})
    
    # 4. 统计成功数量
    $successCount = (Select-String -Path $latestLog.FullName -Pattern "\[SUCCESS\]").Count
    Write-Host "成功操作：$successCount" -ForegroundColor Green
    
    # 5. 显示最后 10 行
    Write-Host "`n=== 最后 10 行日志 ===" -ForegroundColor Cyan
    Get-Content $latestLog.FullName -Tail 10
    
    # 6. 显示所有错误
    if ($errorCount -gt 0) {
        Write-Host "`n=== 错误详情 ===" -ForegroundColor Red
        Select-String -Path $latestLog.FullName -Pattern "\[ERROR\]" -Context 2 | Select-Object -First 5
    }
    
    # 7. 检查截图
    $screenshotCount = (Get-ChildItem screenshots\*.png | Measure-Object).Count
    Write-Host "`n截图数量：$screenshotCount" -ForegroundColor Green
} else {
    Write-Host "未找到日志文件，请先运行抢购脚本" -ForegroundColor Red
}
```

运行:
```powershell
.\quick-diagnose.ps1
```
