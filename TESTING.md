# 🧪 快速测试指南

## 一句话测试

```bash
# 运行完整测试套件（推荐）
npm run test:login-adapter
```

---

## 三个测试场景

### 1️⃣ 测试未登录场景
```bash
npm run test:login-adapter:not-logged
```
**需要**: 先退出智谱 AI 账号  
**预期**: 脚本等待你手动登录

### 2️⃣ 测试已登录场景
```bash
npm run test:login-adapter:logged
```
**需要**: 先登录智谱 AI 账号  
**预期**: 脚本正常找到购买按钮

### 3️⃣ 测试按钮识别
```bash
npm run test:login-adapter:buttons
```
**需要**: 无（任何状态都可以）  
**预期**: 显示所有找到的按钮文本

---

## 测试流程（推荐）

### 步骤 1: 运行完整测试
```bash
npm run test:login-adapter
```

### 步骤 2: 根据提示操作

测试会依次执行：

1. **未登录测试** → 会提示你登录
   - 在打开的浏览器中登录账号
   - 观察测试是否自动继续

2. **已登录测试** → 应该直接通过
   - 因为刚才已登录
   - 检查是否找到按钮

3. **按钮识别测试** → 显示按钮详情
   - 查看按钮文本
   - 查看按钮状态

### 步骤 3: 查看结果

```
✅ 未登录场景：通过
✅ 已登录场景：通过
✅ 按钮识别测试：通过

🎉 所有测试通过！
```

---

## 如果测试失败

### 失败：未登录场景显示"当前已登录"
**解决**: 
1. 访问 https://www.bigmodel.cn/
2. 手动退出登录
3. 重新运行：`npm run test:login-adapter:not-logged`

### 失败：已登录场景显示"当前未登录"
**解决**:
1. 访问 https://www.bigmodel.cn/
2. 手动登录
3. 重新运行：`npm run test:login-adapter:logged`

### 失败：找不到按钮
**原因**: 库存已售罄（正常现象）  
**解决**: 第二天 10:00 再测试
