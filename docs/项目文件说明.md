## 📂 logs/ - 日志输出

**自动生成**，不提交到 Git。

- 文件名：`auto-buy_YYYY-MM-DD.log`
- 自动保留最近 50 个文件
- 每次会话约 1-10MB

## 🚀 快速命令

```bash
# 运行优化版
npm start

# 运行精准对时版（推荐）
npm run start:precision

# 运行标准版
npm run start:standard

# 测试脚本
npm test

# 测试日志系统
npm run test:logger
```

## 📝 Git 提交建议

### 应该提交的文件

- ✅ `src/` - 所有源代码
- ✅ `tests/` - 测试文件
- ✅ `docs/` - 所有文档
- ✅ `package.json` - 项目配置
- ✅ `.gitignore` - 忽略规则
- ✅ `README.md` 等根目录文档

### 不应该提交的文件

- ❌ `node_modules/` - 依赖包
- ❌ `logs/` - 日志文件
- ❌ `screenshots/` - 截图文件
- ❌ `package-lock.json` - 依赖锁定（可选）
- ❌ `*.log` - 日志文件
- ❌ `*.png` - 图片文件

---

## 🎯 项目结构优势

1. **清晰的分类** - 代码、文档、测试分离
2. **自动生成文件独立** - logs、screenshots 自动忽略
3. **跨平台兼容** - 忽略各系统临时文件
4. **便于协作** - 结构清晰，易于理解
5. **安全性** - 敏感文件自动忽略
